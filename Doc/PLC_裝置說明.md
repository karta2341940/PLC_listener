# 三菱 PLC 各裝置說明

## 位元裝置（Bit Device）— 只有 ON/OFF

| 裝置 | 名稱 | 典型用途 | 可修改 |
|------|------|----------|--------|
| **X** | 輸入繼電器 | 接實體輸入訊號（按鈕、感測器、光電開關）。**唯讀**，由硬體決定狀態 | ❌ 唯讀 |
| **Y** | 輸出繼電器 | 控制實體輸出（馬達、電磁閥、指示燈）。程式寫入，硬體執行 | ✅ |
| **M** | 內部繼電器 | PLC 程式內部用的旗標，不對外連接，斷電清除 | ✅ |
| **L** | 鎖存繼電器 | 同 M，但**斷電保持**（停電後狀態不消失） | ✅ |
| **F** | 警報繼電器 | 異常旗標，搭配 ANS 指令使用 | ✅ |
| **B** | 鏈接繼電器 | 用於 CC-Link / MELSECNET 網路資料交換 | ✅ |
| **SM** | 特殊繼電器 | 系統保留，例如 SM400=常時ON、SM401=常時OFF、SM0=運算錯誤 | ❌ 系統決定 |

---

## 字元裝置（Word Device）— 存數值（16-bit）

| 裝置 | 名稱 | 典型用途 | 可修改 |
|------|------|----------|--------|
| **D** | 資料暫存器 | 最常用，存整數數值（溫度、計數、參數），斷電清除 | ✅ |
| **R** | 文件暫存器 | 容量大（萬筆以上），存配方、歷史資料，斷電保持 | ✅ |
| **W** | 鏈接暫存器 | 用於 CC-Link / MELSECNET 網路數值交換 | ✅ |
| **TN** | 計時器現在值 | 計時器當前計數值（單位 ms） | ✅ |
| **TC** | 計時器接點 | 計時器是否到達設定值（位元） | ❌ 邏輯決定 |
| **CN** | 計數器現在值 | 計數器當前計數值 | ✅ |
| **CC** | 計數器接點 | 計數器是否到達設定值（位元） | ❌ 邏輯決定 |
| **SD** | 特殊暫存器 | 系統資訊，例如 SD0=當前錯誤碼、SD315=通信保留時間 | ❌ 系統決定 |

---

## 專案對應關係
```
實體世界              PLC 裝置         SLMP 代碼
─────────────────────────────────────────────
按鈕按下      →  X20 = ON    →  0x9C, addr=0x20
感測器偵測    →  X21 = ON    →  0x9C, addr=0x21
馬達啟動      →  Y30 = ON    →  0x9D, addr=0x30
溫度數值      →  D100 = 250  →  0xA8, addr=100
生產計數      →  D200 = 1500 →  0xA8, addr=200
```

---

## 位址格式注意

| 裝置 | 位址進位 | 範例 |
|------|----------|------|
| X、Y、B | **16 進位** | X20 → addr=`0x20`，X1F → addr=`0x1F` |
| M、L、D、R、W | **10 進位** | M100 → addr=`100`，D50 → addr=`50` |

---

## 可修改性總覽
```
✅ 可修改：Y、M、L、F、B、D、R、W、TN、CN
❌ 不可修改：
   X          → 由硬體決定
   SM / SD    → 由系統決定
   TC / CC    → 由計時器/計數器邏輯決定
```
