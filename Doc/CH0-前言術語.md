# SLMP 參考手冊 — 前言・安全注意事項・術語

> 本文件整理自 SLMP（Seamless Message Protocol）參考手冊，適用三菱電機 MELSEC 可程式控制器系列產品。

---

## 安全注意事項

> 使用之前請務必閱讀

在使用本產品之前，請仔細閱讀本手冊，同時在充分注意安全的前提下正確地操作。

安全注意事項分為 **警告** 和 **注意** 二個等級：

- ⚠️ **警告**：操作錯誤時有引發死亡或重傷的危險。
- ⚠️ **注意**：操作錯誤時有引發輕傷或財產損失的危險。  
  注意根據情況不同，即使「注意」等級也有可能引發嚴重後果。

對兩級注意事項都須遵照執行，請妥善保管本手冊以備查閱，並應將本手冊交給最終用戶。

### 設計注意事項

**警告**

- 將外部設備連接到 SLMP 對應設備上，對運行中的可程式控制器進行控制（資料更改）時，應在程式中組態**互鎖電路**，確保整個系統始終都會安全運行。
- 在對運行中的可程式控制器執行程式更改、參數更改、強制輸出、運行狀態更改（狀態控制）時，應仔細閱讀手冊並充分確認安全之後再進行操作。操作錯誤可能導致機械損壞或引發事故。
- 從外部設備對遠端的可程式控制器進行控制時，由於資料通信異常可能不能立即採取措施。應在程式中組態互鎖電路的同時，預先確定發生資料通信異常時系統的處理方法。
- 在 SLMP 對應設備及智慧功能模組的緩衝記憶體中，**請勿對「系統區域」或「禁止寫入區域」進行資料寫入**。
- 在 SLMP 對應設備及智慧功能模組的輸出信號中，**請勿對「禁止使用」的信號進行輸出（ON）操作**。若違反此規定，有可能導致可程式控制器系統誤動作。

### 啟動·維護注意事項

**注意**

- 將外部設備連接到運行中的 SLMP 對應設備或其它站的 CPU 模組上進行線上操作（程式更改、參數更改、強制輸出、運行狀態更改）時，應仔細閱讀手冊並充分確認安全之後再進行操作。

## 前言

本手冊讓用戶瞭解使用 SLMP（Seamless Message Protocol）時所需要的：

- 對應設備
- 訪問範圍
- 通信步驟
- 報文格式

在使用之前應熟讀本手冊，在充分瞭解 SLMP 的基礎上正確地使用本產品。  
將本手冊中介紹的程式示例應用於實際系統的情況下，應充分驗證物件系統中不存在控制方面的問題。

## 術語

| 術語 | 說明 |
|------|------|
| A系統CPU | 冗餘系統中，對熱備電纜 A系統連接器進行了連接的一側的 CPU 模組。 |
| B系統CPU | 冗餘系統中，對熱備電纜 B系統連接器進行了連接的一側的 CPU 模組。 |
| CC-Link IE TSN | 使用擴展乙太網路（1000BASE-T）的 TSN（Time Sensitive Networking）標準的網路，與 CC-Link IE 控制網路及 CC-Link IE 現場網路相比高性能且高功能。CC-Link IE TSN 的模組支持 SLMP。 |
| CC-Link IE 控制網路 | 使用乙太網路（1000BASE-SX）光纖電纜或乙太網路（1000BASE-T）的大規模控制器分散控制最佳的網路。 |
| CC-Link IE 現場網路 | 使用乙太網路（1000BASE-T）的高速且大容量的開放現場網路。 |
| 工程工具 | MELSEC 可程式控制器軟體包的別稱。 |
| 回應報文 | 對於請求報文，SLMP 對應設備發送到個人電腦及顯示器等的外部設備中的處理結果報文。 |
| 管理CPU | 控制各輸入輸出模組、智慧功能模組的 CPU 模組。 |
| 請求報文 | 從個人電腦及顯示器等外部設備向 SLMP 對應設備發送的訪問請求報文。 |
| SLMP 對應設備 | 支持 SLMP 通信的設備。 |

| 總稱/簡稱 | 說明 |
|----------|------|
| CC-Link IE TSN 主站·本地站模組 | 表示 RJ71GN11-T2、RJ71GN11-EIP（CC-Link IE TSN 部）。 |
| CC-Link IE 控制網路搭載模組 | RJ71GP21-SX、RJ71GP21S-SX、QJ71GP21-SX、QJ71GP21S-SX 型 CC-Link IE 控制網路模組及使用 CC-Link IE 控制網路功能時的 RJ71EN71、RnENCPU 的總稱。 |
| CC-Link IE 現場網路起始模組 | LJ72GF15-T2 型 CC-Link IE 現場網路起始模組的簡稱。 |
| CC-Link IE 現場網路主站·本地站搭載模組 | RJ71GF11-T2、QJ71GF11-T2、LJ71GF11-T2 型 CC-Link IE 現場網路主站·本地站模組及使用 CC-Link IE 現場網路功能時的 RJ71EN71、RnENCPU 的總稱。 |
| CC-Link IE 現場網路遠端起始模組 | RJ72GF15-T2 型 CC-Link IE 現場網路遠端起始模組的簡稱。 |
| 乙太網路搭載模組 | 搭載乙太網路介面的 MELSEC 模組（如 RJ71EN71 等）。 |
| CPU 模組 | MELSEC iQ-R 系列、MELSEC-Q 系列、MELSEC-L 系列等各型號 CPU。 |
