# 第1章 SLMP 的作用

## 概述

**SLMP（Seamless Message Protocol）** 是乙太網路中使用的通信協定（控制步驟）。  
可以通過外部設備對 SLMP 對應設備及其連接的可程式控制器進行訪問。

---

## 1.1 通過外部設備監視系統

```
外部設備 (1)  ──[請求報文(3)]──►  SLMP對應設備 (2)
              ◄──[回應報文(4)]──
```

通過從使用乙太網路連接的**外部設備(1)** 向 **SLMP 對應設備(2)** 發送**請求報文(3)**，並接收**回應報文(4)**：

- 可進行軟元件的**讀取**（Read）
- 可進行軟元件的**寫入**（Write）
- 可執行 SLMP 對應設備的**復位**（Reset）等

> 詳細陳述式操作請參閱第5章「陳述式」。

---

## 1.2 連接 MC 協定中使用的外部設備

SLMP 的 **3E幀** 或 **4E幀** 的報文格式與 MC 協議的 **QnA 系列 3E幀或 4E幀** 相同。  
可以將 MC 協定中使用的外部設備原樣不變地連接到 SLMP 對應設備上。

> 相關參照：附錄2「MC協議與SLMP的陳述式對應表」

---

## 1.3 與其它網路的無縫訪問

```
外部設備 (1) ──► SLMP對應設備 (2) ──► 其它網路的模組 (3)
              乙太網路
```

如果使用 SLMP，可以通過外部設備(1)經由 SLMP 對應設備(2)，對**其它網路的模組(3)進行無縫訪問**。

> 相關參照：第2.3節「訪問範圍及可其它站訪問的模組」

---

## 1.4 通過通信協議支援功能進行通信

可以使用工程工具的**通信協定支援功能**簡單地進行 SLMP 的通信。

```
乙太網路搭載模組 (1) ──[請求報文(3)]──► SLMP對應設備 (2)
                     ◄──[回應報文(4)]──
```

> ⚠️ **注意**：通信協定支援功能在 **CC-Link IE TSN 主站·本地站模組** 及 **RD78G** 中**無法使用**。

---

## 1.5 通過專用陳述式進行控制

可以從 CC-Link IE TSN 主站·本地站模組或 CC-Link IE 現場網路主站·本地站搭載模組，  
通過**專用陳述式**對 SLMP 的請求資料與回應資料進行發送接收，並控制同一網路上的 SLMP 對應設備。

### 控制 SLMP 對應設備的專用陳述式

| 陳述式 | 適用網路 | 備註 |
|--------|----------|------|
| `SLMPSND` | CC-Link IE TSN | — |
| `SLMPREQ` | CC-Link IE 現場網路 | 僅 MELSEC iQ-R 系列支援 |

> 詳細內容請參閱《MELSEC iQ-R 程式手冊（模組專用陳述式篇）》。
